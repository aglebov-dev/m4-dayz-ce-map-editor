# Стилизованная иконка для приложения "m4 dayz ce map editor"

## Созданные файлы

1. **app_icon.png** (42 KB, 512x512)
   - PNG версия иконки с высоким разрешением
   - Используется как источник для генерации ICO

2. **app_icon_high.ico** (471 bytes)
   - Основная ICO иконка приложения
   - Содержит 7 размеров: 16x16, 24x24, 32x32, 48x48, 64x64, 128x128, 256x256
   - Используется в сборке приложения (pysidedeploy.spec) и в коде

## Особенности дизайна

Иконка создана на основе существующего логотипа `logo.jpg` с добавлением:
- Темно-сине-серый фон с полупрозрачным логотипом на заднем плане
- Надпись "m4 dayz" в центре с градиентом от оранжевого к желтому
- Тень и легкий контур для лучшей читаемости
- Стилизация в духе DayZ (милитари-техническое направление)

## Интеграция

Иконка интегрирована в приложение:

1. **Конфигурация сборки** (`src/pysidedeploy.spec:19`)
   - Установлена новая иконка вместо стандартной PySide6

2. **Код приложения** (`src/app.py:17-20`)
   - Иконка устанавливается для QApplication

3. **Главное окно** (`src/light/main_window.py:42-46`)
   - Иконка устанавливается для главного окна

## Как обновить иконку

Для обновления иконки:
1. Замените файл `logo.jpg` в корне проекта
2. Запустите `python create_icon.py` для генерации новой иконки
3. Убедитесь, что код сборки использует правильный путь к файлу

## Скрипты

- `create_icon.py` - основной скрипт для создания иконки из `logo.jpg`
- `create_high_quality_ico.py` - улучшенная генерация ICO со всеми размерами
- `check_icon.py` - проверка созданных файлов иконки
