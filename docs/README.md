# M4 DayZ CE Map Editor

Редактор зон Central Economy (DayZ). Фокус: работа с `areaflags.map`, продолжение
проекта CE Tool (BI), экспорт в проект BI, диф.

## Запуск
`run.bat` (использует локальную `.venv`; собрать её — `rebuild-env.bat`).

## Идея
Данные грузятся через **провайдеров** (файловая система или SFTP — корень сервера DayZ).
Проект = набор выбранных файлов + снапшот для сравнения (диф). Инструмент без нужных
файлов блокируется с подсказкой.

План — `PLAN.md`. Знания о форматах — `docs/knowledge.md`.
